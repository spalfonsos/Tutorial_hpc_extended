# 🚀 HPC Extended Tutorial

This repository provides the summary of what is going to be explained in the tutorial classess for bcp.

## 📘 What you’ll learn
- How to log in securely using SSH
- Structure in remote clusters
- Some important commands in ssh
- How to create environments and start interactive sessions
- Use interactive session for runing jupyter notebook
